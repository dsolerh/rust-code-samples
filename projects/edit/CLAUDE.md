## Project Overview

Lightweight terminal-based text editor for programming languages, written in Rust (edition 2024).

## Build & Run Commands

- **Build**: `cargo build`
- **Run**: `cargo run`
- **Run (release)**: `cargo run --release`
- **Check (fast compile check)**: `cargo check`
- **Test**: `cargo test`
- **Run single test**: `cargo test <test_name>`
- **Lint**: `cargo clippy`
- **Format**: `cargo fmt`
